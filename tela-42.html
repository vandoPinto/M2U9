Métodos de programação

Existem alguns métodos para realizar a programação. Marin (2003) e Blatt, Campos e Becker (2015) apresentam alguns desses métodos de programação para medicamentos. Contudo, esses métodos podem ser empregados como critérios técnicos eficientes para programar outros materiais e insumos de saúde.

Blatt, Campos e Becker (2015) destacam que existe uma tradição, em grande parte das instituições, de utilizarem o método de consumo histórico para realizar a programação de medicamentos, mas que esse não é o único método que está disponível para ser utilizado e que o uso de um único critério técnico pode contribuir para que erros contínuos sejam mantidos; por essa razão, é recomendável utilizar vários métodos combinados, a fim de alcançar uma programação mais adequada.
