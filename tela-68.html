TÓPICO 3

ARMAZENAMENTO, DISTRIBUIÇÃO E DISPENSAÇÃO DE MEDICAMENTOS

Definir o layout não é uma tarefa intuitiva. Você deve obedecer à dinâmica de movimentação dos materiais no espaço físico.

Então, aquilo que é considerado como layout de um almoxarifado trata-se de um ordenamento de fluxos, pessoas, equipamentos e materiais que seguem uma disposição dentro do padrão de racionalidade (FENILI, 2015; BERTAGLIA, 2006).

Alguns autores, como Muther (1978), definem que o ponto decisivo para definir um layout que seja satisfatório começa pela definição das características dos itens que vão ser acondicionados nos almoxarifados.
